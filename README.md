# 通用剧本工作流：三分钟剧本 + 15秒 LibTV

这是通用剧本工作流备份仓库，不再绑定 SCRAPBRO / 赛铁坡单一项目。

仓库现在包含两类入口：

1. 通用三分钟剧本工作流：把一个剧情点做成可观看、可拍摄、可剪辑、可持续连载的 3 分钟剧本单元。
2. 通用剧本 15 秒 LibTV 工作流：把已经稳定的剧本继续拆成 15 秒一段的 LibTV / 里布视频提示词、图片绑定卡、缺图补图流程和多格分镜参考图流程。

## 先读哪个

如果要先写或优化三分钟剧本，先读：

```text
1. 30_通用三分钟剧本工作流_three_minute_script_workflow.md
```

如果已经有三分钟剧本，要继续做 15 秒 LibTV / 里布视频提示词，按顺序读取：

```text
1. 00_15秒工作流强制启动卡.md
2. 00_通用剧本15秒工作流_V1.md
3. 00_开新窗口_通用剧本15秒工作流完整启动提示词模板.md
4. WORKFLOW_INDEX.md
```

然后根据 `WORKFLOW_INDEX.md` 继续读取 `workflows/` 里的核心规则。

## 三分钟剧本核心流程

```text
启动输入
-> 输入不足自动补全
-> 题材适配
-> 三分钟结构
-> 单集生产卡
-> 正式剧本正文初稿
-> 顶级编剧 / 顶级导演 / 顶级剪辑三视角复盘优化
-> 连续性检查
-> 审稿闸门
-> 通过 / 退回判定
-> 按用途输出
```

## 15秒 LibTV 核心流程

```text
版本冻结
-> 顶级导演 / 顶级编剧 / 顶级剪辑预审
-> 直接产出 6-8 个细分镜头草案
-> 图片绑定卡
-> 缺图补图门
-> 连续性四帧
-> 唯一主视觉任务
-> 第一份正式输出：完整 15 秒 LibTV 提示词
-> 所有主锁定图补齐并 QA 后，重新输出新版完整提示词
-> 多格分镜参考图专项流程
-> 分镜图 QA 通过后回填最终上传顺序
```
