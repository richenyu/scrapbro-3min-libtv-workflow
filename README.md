# 通用剧本 15 秒 LibTV 工作流

这是最新版“通用剧本 15 秒工作流”备份仓库。

它不再绑定 SCRAPBRO / 赛铁坡单一项目，可以用于任意剧本，把某一段剧情稳定转成 LibTV / 里布可直接使用的 15 秒提示词、图片绑定卡、缺图补图流程和多格分镜参考图流程。

## 先读哪个

新窗口或新电脑恢复工作流时，按顺序读取：

```text
1. 00_15秒工作流强制启动卡.md
2. 00_通用剧本15秒工作流_V1.md
3. 00_开新窗口_通用剧本15秒工作流完整启动提示词模板.md
4. WORKFLOW_INDEX.md
```

然后根据 `WORKFLOW_INDEX.md` 继续读取 `workflows/` 里的核心规则。

## 核心流程

```text
版本冻结
-> 顶级导演 / 顶级编剧 / 顶级剪辑预审
-> 直接产出 6-8 个细分镜头草案
-> 图片绑定卡
-> 缺图补图门
-> 连续性四帧
-> 唯一主视觉任务
-> 第一份正式输出：完整 15 秒 LibTV 提示词
-> 所有主锁定图补齐并 QA 后，重新输出新版完整提示词
-> 多格分镜参考图专项流程
-> 分镜图 QA 通过后回填最终上传顺序
```
