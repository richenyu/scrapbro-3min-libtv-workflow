# 15秒分镜头参考图工作流

用途：以后只要需要把某一段完整 15 秒分镜头提示词生成“分镜头参考图 / 六格分镜图 / 九格参考图”，必须先执行本工作流。

核心原则：

```text
先把 15 秒提示词写对，再检查锁定图是否完整，再生成分镜头参考图。
参考图是完整分镜的视觉落实，不是倒推分镜的起点。
旧错图不能继续作为参考图污染下一版。
如果本段承接上一段 15 秒，不再准备上一段最后一帧 / 尾帧承接图 / 伪真尾帧图；只先确认前置视频节点、连续性四帧、声音桥和动作方向，再生成本段多格分镜参考图。
```

## 第一步：顶级导演 + 顶级编剧 + 顶级剪辑思维重新优化完整 15 秒提示词

目标：先确认这一段 15 秒本身是因果成立、人物选择成立、可拍、可剪、可接的完整视频单元。

必须完成：

1. 明确导演判断：本段最重要的一个画面动作是什么，这一段只解决一个主视觉任务。
2. 明确编剧判断：本段为什么必须发生，人物为什么这样选择，选择后改变了什么。
3. 明确信息取舍：哪些剧情信息必须在本段讲清，哪些必须删掉或留到下一段。
4. 明确开场钩子：前 1-2 秒用什么画面 / 声音抓住观众。
5. 明确上一视频最后一帧：本段第一帧从哪里接，不重新演上一段已经完成的动作。
6. 明确本段结尾锚点：最后一帧能自然接下一段。
7. 明确镜头节奏：本段是追击、急刹、对峙、突袭、反击、取物、转场还是压迫建立。
8. 明确剪辑重心：哪些镜头要快切，哪些镜头要停半秒让观众看懂。
9. 明确情绪曲线：紧张、缓冲、爆点、余波分别落在哪几秒。
10. 明确对白功能：如果有对白，只推动人物关系、选择或危机判断，不做过量解释。
11. 明确角色数量：核心视觉角色控制在 1-3 个，其他人不用正面出镜。
12. 明确关键实物：每个关键道具开始在哪里、谁碰过、结束在哪里、下一段怎么继承。
13. 直接输出 6-8 个 15 秒细分镜头草案：根据剧情节奏自由拆分，不机械平均，不只写剧情梗概。
14. 写清人物表情、身体状态、手部动作、眼线关系、前景/中景/背景。
15. 写清声音桥、声音剪点、穿帮禁止项和 AI 视频可拍性检查。
16. 如果本段是追逐 / 逃亡，必须先写“追逐队列硬锁”：最前方是逃跑路线，中间是正在逃跑的主角 / 伙伴，最后方是追击者 / Boss。这个顺序不能改变，追击者不能出现在前方、侧面并排、出口处或下一段方向。
17. 如果本段是废弃场景，必须写清废弃程度和时代物件：破、旧、脏、空、停摆；禁止现代车流、现代停车场、干净运营场地、整齐新货箱、正常工作设备。
18. 如果本段出现英文对白，必须执行中文提示词规则：除角色真正说出口的英文对白外，所有提示词说明、锁图命令、场景描述、声音描述、镜头调度、负面约束全部使用中文；英文对白后保留中文翻译，中文不配音、不出字幕。

输出要求：

```text
先输出或内部形成 6-8 个细分镜头草案，再输出或内部形成完整 15 秒提示词。
细分镜头草案必须覆盖完整 00:00-00:15，每个镜头都要写清画面动作、编剧功能、剪辑点、声音点和承接关系。
没有完整 15 秒提示词，不进入第二步。
```

## 第二步：参考图完整性检查与补图

目标：在生成分镜头参考图前，确认本段需要的图片素材够不够、准不准。

必须检查：

1. 前置视频节点是否存在；如果不存在，连续性四帧、声音桥和动作方向是否已经写清。
2. 本段所有正面出镜角色是否有锁脸图、全身图或足够可靠的角色锁定图。
3. 本段关键道具是否有锁定图，并且尺寸、形状、材质、方向都清楚。
4. 本段场景是否有场景锁定图；如果没有，是否能用上一帧和文字稳定锁定。
5. 本段六格/九格参考图是否是新生成的正式图，还是旧流程/旧错图。
6. 如果涉及眼睛、左右手、机械义肢、道具体量、门/抽屉/箱盖开合状态等易错点，必须实际打开锁定图看一遍，不能只看文件名。

必须把参考图分成三类：

```text
已有可直接使用：
- ...

已有但需谨慎：
- ...

缺失必须补：
- ...
```

缺图处理：

```text
如果缺人物锁定图、关键道具图或场景图，先补齐。
如果缺前置视频节点，不补上一段尾帧图片，只用连续性四帧和文字约束重建承接。
补图完成后，回到第一步，重新检查完整 15 秒提示词是否需要因新图调整。
```

## 第二点五步：重出新版完整 15 秒提示词

多格分镜参考图不是普通补图。所有主锁定图补齐并通过 QA 后，不能直接进入生图，必须先重新输出一版“整合全部锁定图的新版本完整 15 秒提示词”。

新版完整提示词必须整合：

```text
最新图片绑定卡和 @图片N 顺序
全部已确认人物 / 伙伴 / 副角色 / 场景 / 敌人 / 怪物 / Boss / 比例 / 载具 / 道具锁定图
连续性四帧
顶级导演 / 顶级编剧 / 顶级剪辑预审结果
本段唯一主任务
声音锁定和字幕规则
动作地理和空间轴线
每个细分镜头的时间段、动作事件、角色位置、敌人 / 怪物 / Boss 来源、打击结果和声音重点
结尾承接
防错审计
```

硬规则：

```text
不能拿补图前的旧提示词直接生成多格分镜参考图。
不能只把新图片文件名追加到旧提示词末尾。
必须让新版完整提示词和所有锁定图互相对应，再进入第三步。
```

上一段尾帧 / 伪真图取消规则：

```text
15 秒工作流不再加入上一段最后一帧 / 尾帧承接图 / 伪真尾帧图。
不生成、不截取、不预留、不命名、不上传任何上一段尾帧图片。
上一段落点、本段起点、动作方向、角色位置、场景连续和道具状态，只写进连续性四帧、声音桥、动作地理和提示词正文。
多格分镜参考图仍然放在最后一个图片槽；常规 7 张锁定图后，@图片8 就是多格分镜参考图。
```

旧错图隔离规则：

```text
如果某一版分镜头参考图已经出现人物眼睛方向错误、道具体量错误、角色重复、场景跳变、道具变形等问题，
下一版生成时不能再把这张旧错图列入上传顺序，也不能在生成提示词里继承它。
必须明确写：忽略上一版错误参考图，只参考首帧图、人物锁定图、道具锁定图和场景锁定图。
确认废掉的参考图必须删除，不允许继续留在素材目录里，也不要用“不作为正式参考”“废稿”“旧版”这类文件名长期保留。
删除范围只限已经确认错误的分镜参考图、临时生成图、旧错图和废稿；不要删除正式锁定图、角色图、场景图、敌人 / 怪物 / Boss 图、道具图和已经通过审核的历史素材。
删除前必须核对绝对路径，确保文件确实属于本轮要清理的参考图废稿；删除后再生成新版参考图，避免下一轮误把废图作为输入参考。
```

锁定图漂移否决规则：

```text
分镜参考图只锁动作顺序、镜头节奏、站位、方向、起点和落点，不能重新设计人物、伙伴、载具、敌人 / 怪物 / Boss、道具或场景。
只要分镜图里的角色、伙伴 / 副角色、载具、敌人 / 怪物 / Boss 物种、关键道具状态明显不像正式锁定图，即使动作顺序正确，也必须判定为废稿并删除。
如果伙伴 / 副角色有单眼方向、机械结构、身体比例、宠物物种等硬锁，分镜图不得把它画成新角色、错误结构、错误眼位或不同物种。
如果载具已有正式锁定图，分镜图不得把它改成普通摩托、跑车、悬浮车、不同后箱结构或不同车身比例。
如果敌人 / 怪物 / Boss 已有正式锁定图，分镜图不得把它改成旧版本敌人、错误 Boss、纯机器人、站立人形或其它物种。
如果关键道具有“残片 / 完整体 / 开启 / 关闭”等状态，分镜图不得改变状态；道具状态一错，整张参考图判废。
```

污染窗口停止规则：

```text
如果某张错误参考图已经在当前聊天窗口、当前生图会话或当前生图工具上下文里出现过，只删除硬盘文件还不够。
当前窗口已经看过错误图，继续在同一窗口里重生，仍可能受到旧图视觉上下文污染。
一旦出现连续参考图漂移，必须停止在当前窗口继续生成该 Part 参考图；先删除项目目录和生成缓存里的错误图，再切换到干净窗口 / 干净生图会话。
干净会话只允许加载正式锁定图、当前完整提示词和参考图一致性生图提示词；禁止加载任何旧参考图、废稿图、过程截图、拼板输入图和含旧参考图的文档截图。
重新生成前必须运行污染检查：项目目录无旧 Part 参考图、上传包无旧图片位、生成缓存无本轮废图、文档无旧图路径。
```

动作类参考图额外规则：

```text
如果本段是打斗、追击、怪物突袭或群体压迫，六格/九格参考图必须表现敌人持续移动。
敌人 / 怪物不能像海报摆拍一样站在原地，不能停在角色面前等待攻击，不能围成静态圆圈。
每一格里的敌人要么正在冲刺、扑击、爬越、撞开障碍、被打飞、滑出攻击线，要么在背景虚焦中快速靠近。
如果主角、伙伴或其他角色有扫描、蓄力、预判动作，敌人仍然必须处在移动中；参考图要表现“截击移动目标”，不是“打静止靶”。
追逐参考图必须表现固定队列：逃跑路线在前，逃跑者在中，追击者 / Boss 在后。禁止把追击者放到逃跑者前方、侧面并排、出口处或下一段方向。
带链条、绳索、触手、管线的敌人 / 怪物必须控制数量和用途：只画剧情需要的那一条或少数几条关键链条，不能画成满身铁链、铁链团、绳网或无意义缠绕怪。链条必须服务动作接触点。
```

## 第三步：基于新版完整提示词和已确认参考图生成分镜头参考图

目标：把已经通过检查并重新输出的新版 15 秒提示词视觉化为一张可用的多格分镜参考图。

生成前必须写清：

1. 实际参考图上传顺序：`@图片1 / @图片2 / @图片3 ...` 必须和 LibTV 或图像生成工具实际上传顺序一致。
2. 每张图的用途：人物锁定、道具锁定、场景锁定、动作分镜；上一段尾帧 / 伪真图不再作为图片用途。
3. 本段每一格对应的时间段和镜头事件。
4. 人物一致性：脸、眼睛方向、服装、身体比例、机械结构、手套/武器状态。
5. 道具一致性：尺寸、形状、颜色、材质、位置、开合状态和下一段继承方式。
6. 场景一致性：主方向、光线、前景/中景/背景、固定地标。
7. 色彩饱和度一致性：保持克制，特效只做小面积点缀。
8. 禁止项：字幕、标题、UI、水印、旧错图继承、角色重复、道具放大、方向镜像等。

还必须写清：

```text
多格分镜参考图一致性生图提示词。
每一张输入参考图锁什么。
每一格对应新版完整提示词中的哪个时间段和镜头事件。
分镜图只能学习动作顺序、镜头节奏、站位、方向、起点、落点和结尾承接。
分镜图不能改写人物脸、伙伴特殊结构、载具结构、敌人 / 怪物 / Boss 物种、真实比例、道具状态和场景地标。
```

分镜头参考图标准：

```text
单张 3列 x 2行六格图，或用户指定的九格图。
从左到右、从上到下读取。
覆盖完整 00:00-00:15。
每格是一个清楚镜头事件。
无字幕、无标题、无对白气泡、无 UI、无水印。
```

禁止把参考图做成锁定图裁切拼贴板：

```text
正式分镜头参考图必须是电影分镜式连续画面，不是把人物锁定图、敌人 / 怪物 / Boss 锁定图和场景图裁切后贴到背景上的拼贴图。
拼贴图、裁切锁定图、带白底块的人物图、带设定表文字的截图、角色设定表局部裁剪，都不能作为正式六格分镜参考图交付，也不能进入 LibTV 上传顺序。
如果当前工具不能真实上传并引用人物锁定图、敌人 / 怪物 / Boss 锁定图和场景锁定图参与生成，就不要强行生成正式参考图；必须停下并说明需要在支持参考图输入的平台里生成。
```

## 第四步：生成后检查、保存和输出链接

目标：生成图不能直接交付，必须先按锁定图复核。

必须检查：

1. 是否确实是单张六格/九格参考图。
2. 是否按时间顺序读成一个完整 15 秒。
3. 是否没有换脸、换装、换身体比例。
4. 是否没有角色重复、背景人升级成主角。
5. 是否没有场景跳变。
6. 是否没有把关键道具放大、缩小、换形状、换材质或换位置。
7. 是否符合本段特殊锁定项，例如机械眼方向、伙伴特殊结构、关键道具体量、手套是否仍戴在手上。
8. 是否没有字幕、标题、UI、水印。
9. 是否和完整 15 秒提示词一致，而不是另起一段戏。

如果不合格：

```text
不要进入 LibTV 视频生成。
不要把错误图继续列入下一版参考图。
把错误点写进下一版生成提示词的硬约束，并重新从第三步生成。
如果错误图是本轮临时生成图、旧错图或废稿，必须直接从本地删除；删除前只核对该图确实不是正式锁定图或已通过审核素材。
不要保留带“不作为正式参考”“废稿”“旧版文字直生”等命名的错误参考图；这些文件会污染下一轮生成。
```

如果合格：

```text
保存到 `参考图` 文件夹。
文件名使用：
PartXX_六格分镜参考图_[本段名]_[关键修正点]_V数字.png

不再输出上一段最后一帧 / 尾帧承接图 / 伪真尾帧图的文件名建议，也不预留上传槽位。

输出可打开链接。
如果用户需要上传顺序，同时按 @图片 顺序输出所有参考图链接。
```

## 执行提醒

以后遇到以下任务时，直接调用本工作流：

```text
生成 15 秒分镜头参考图
生成六格分镜参考图
根据完整版提示词生成参考图
重做这一 Part 的分镜图
按锁定图生成分镜图
参考图跑偏后重新生成
```

一句话流程：

```text
先优化完整 15 秒分镜词，再实际看锁定图是否完整，缺图先补图，最后才生成分镜参考图；错误旧图必须隔离，不准继续污染新图。
```
