# 00_15秒工作流强制启动卡

更新时间：2026-06-21

## 用途

这是通用剧本 / LibTV / 里布 15 秒工作流的新窗口强制入口。

本卡先执行通用主流程，再根据当前剧本加载项目适配层。SCRAPBRO、赛铁坡、RAM、酸翼刃兽等只属于当前项目示例或项目专用执行包，不是通用流程的固定角色。

只要用户提到：

```text
15秒工作流
15秒提示词
LibTV / 里布
生成分镜参考图
八格分镜 / 六格分镜
上传图包
某个 Part / 第几段 / 当前这15秒
通用剧本 15 秒工作流
```

Codex 必须先执行本启动卡。没有完成本卡，不允许直接输出最终提示词、分镜图提示词或上传图包。

## 第 0.5 步：顶级导演 / 顶级编剧 / 顶级剪辑预审卡

版本冻结后，进入图片绑定前，必须先用顶级电影导演、顶级编剧和顶级剪辑师思维预审并优化这一段 15 秒完整提示词。

目的：

```text
先判断这 15 秒为什么发生、怎么拍、怎么剪、怎么接，再决定图片绑定和完整提示词。
不是机械拆 8 格，不是按剧情文字平均切段。
```

必须先列出，并直接变成 6-8 个细分镜头草案：

```text
导演判断：本段最重要的一个画面动作是什么？
编剧判断：本段的因果、人物选择、冲突压力和结果变化是否成立？
信息取舍：哪些剧情信息必须在本段讲清，哪些必须删掉或留到下一段？
开场钩子：前 1-2 秒用什么画面 / 声音把观众抓住？
镜头节奏：这 15 秒是追击、急刹、对峙、突袭、反击、取物、转场还是压迫建立？
剪辑重心：哪些镜头必须快切，哪些镜头必须停半秒让观众看懂？
动作可读性：关键接触点、受力方向、落点是否清楚？
情绪曲线：紧张、缓冲、爆点、余波分别落在哪几秒？
对白功能：如果有对白，是否只推动人物关系、选择或危机判断，而不是解释过量？
声音剪点：上一段尾音如何进来，本段尾音如何带到下一段？
结尾承接：最后 1-2 秒留给下一段的画面入口或声音入口是什么？
删减判断：哪些信息不能塞进本段，必须留到下一段？
```

预审阶段必须输出 6-8 个细分镜头草案：

```text
镜头1｜00:00-00:__：
- 画面 / 动作：
- 编剧功能：
- 剪辑点：
- 声音点：
- 承接关系：

镜头2｜00:__-00:__：
- 画面 / 动作：
- 编剧功能：
- 剪辑点：
- 声音点：
- 承接关系：

继续到 6-8 个镜头，必须覆盖完整 00:00-00:15。
```

细分镜头草案硬规则：

```text
必须是 6-8 个镜头，不能少于 6 个，不能超过 8 个，除非用户明确指定。
每个镜头只处理一个清楚动作事件或一个清楚信息点。
时间段必须连续覆盖 00:00-00:15，不能漏秒、重叠或机械平均。
每个镜头都要写清画面动作、编剧功能、剪辑点、声音点和承接关系。
后续完整 LibTV 提示词里的 15 秒细分镜头，必须以这版预审细分镜头草案为底稿进行校正和扩写。
```

硬规则：

```text
导演 / 编剧 / 剪辑预审要先产出 6-8 个细分镜头草案；它不替代正式锁定图，不生成分镜参考图。
预审后才能进入图片绑定卡。
如果预审发现本段塞了两个大事件，必须先拆段或删减，再继续。
```

上一段尾帧 / 伪真图取消规则：

## 第 2 步：缺图补图门

每段 15 秒如果缺少必要锁定图，必须先补图。补图发生在多格分镜参考图之前。

必须先检查：

```text
是否缺主角 / 新人物锁定图：
是否缺伙伴 / 副角色 / 对手锁定图：
是否缺当前场景锁定图：
是否缺当前敌人 / 怪物 / Boss 正式锁定图：
是否缺当前敌人 / 怪物 / Boss 比例图：
是否缺载具 / 武器 / 任务道具锁定图：
是否缺当前道具状态图（残片 / 完整体 / 开启 / 关闭）：
```

如果缺图，必须按这个顺序生成：

```text
1. 人物 / 角色锁定图
2. 场景锁定图
3. 敌人 / 怪物 / Boss 正式锁定图
4. 敌人 / 怪物 / Boss 比例图
5. 载具 / 武器 / 任务道具锁定图
6. 道具状态补图
```

补图硬规则：

```text
人物图、场景图、敌人 / 怪物 / Boss 图、比例图、道具图都是主锁定图。
上一段最后一帧 / 尾帧承接图 / 伪真尾帧图已取消，不生成、不上传、不占位。
上一段落点、本段起点、方向和角色状态只写入连续性四帧与提示词正文。
不能用多格分镜图临时替代人物锁定图、场景锁定图、怪物比例图或道具状态图。
缺图时不能为了省事直接生成分镜图。
补图完成后必须更新图片绑定卡，再进入后续步骤。
多格分镜参考图不是补图，不在本步骤直接生成；必须进入“多格分镜参考图专项流程”。
```

小兵 / 杂兵群体差异化规则：

```text
小兵、杂兵、普通感染者、普通小怪群、路人敌人、群演敌人不能锁成同一张脸。
如果需要补小兵 / 杂兵参考图，只锁物种方向、感染风格、服装系统、装甲等级、机械感染比例、颜色规则、武器类型等共同设计语言。
每个个体的脸、头骨轮廓、眼位、嘴部、伤口、腐烂纹理、机械残片位置、体型比例和姿态必须不同。
禁止复制同一张脸、同一个头、同一只怪、同一套表情或同一模型排队。
小兵群体图可以是“同设计语言下的多样个体”，不能变成一个模板怪的复制粘贴。
精英怪、Boss、当前关键敌人才允许用单一正式锁定图锁脸、锁比例、锁外形。
```

补图后必须 QA：

```text
人物是否和既有角色一致。
场景是否和本段地点一致。
敌人 / 怪物 / Boss 是否是当前版本，比例是否正确。
如果是小兵 / 杂兵群，是否做到同设计语言但不同脸、不同头骨轮廓、不同腐烂纹理和不同机械残片位置。
道具状态是否正确。
是否有文字、水印、标题、无关角色。
是否会污染当前段上传槽位。
```

不通过则返工；通过后才能进入多格分镜参考图专项流程。

必须写清：

```text
本段最后一帧：

```

这四帧不清楚时，不允许进入正式提示词。

## 第 4 步：唯一主任务

每个 15 秒只允许一个主视觉任务。

必须写成一句话：

```text
本段只解决：……
```

禁止在一段里同时完成两个大事件。下一段动作不能提前写进来。

## 第 5 步：先写完整提示词，再生成分镜图

第一输出硬规则：

```text
版本冻结卡、导演 / 编剧 / 剪辑预审卡、图片绑定卡、缺图补图门和连续性四帧确认后，第一份正式创作输出必须是“按 LibTV / 里布提示词写作方式优化后的完整 15 秒提示词”。
不能先输出分镜参考图提示词，不能先生成多格分镜图，不能只写剧情梗概。
```

这份完整 15 秒提示词必须先包含：

```text
图片上传顺序
本段唯一主视觉任务
人物一致性
场景一致性
敌人 / 怪物 / Boss 一致性
道具一致性
声音一致性 / 角色声音锁定
字幕规则
动作地理 / 空间轴线
15 秒细分镜头
结尾承接
防错审计
```

图片引用格式硬规则：

```text
提示词正文和上传清单必须使用 LibTV 实际上传顺序：@图片1、@图片2、@图片3……连续编号。
不能在最终提示词正文里写 @角色名、@怪物名、@道具英文名 这类素材标签。
每个 @图片N 必须在图片上传顺序里说明中文名称、用途和原文件名 / 锁定内容。
```

正确顺序：

```text
完整 15 秒提示词定稿
-> 8 个或对应数量细分镜头确认
-> 人物 / 场景 / 敌人 / 怪物 / Boss / 道具锁定图确认
-> 声音一致性 / 角色声音锁定确认
-> 缺图已先补齐并通过 QA
-> 重新输出整合所有锁定图的新版本完整 15 秒提示词
-> 有上一段衔接时，只确认前置视频节点和连续性四帧，不添加上一段尾帧 / 伪真尾帧图片槽位
-> 再进入多格分镜参考图专项流程
-> 分镜图只作为流程参考回填
```

绝对禁止：

```text
先凭概念生成分镜图
再用分镜图倒推 15 秒提示词
```

## 第 6 步：一致性和声音锁定必须显式写入

正式提示词和分镜参考图提示词中，必须同时包含：

```text
人物一致性：
- 主角 / 伙伴 / 关键角色的脸、发型、服装、身体比例、装备、伤损状态、手持道具、站位关系。
- 禁止换脸、换装、无理由变比例、无理由互换位置。
- 如果当前项目有机械眼、义肢、面罩、纹身、伤疤、特殊宠物或 AI 伙伴等硬锁，必须在这里逐条写清。

场景一致性：
- 当前场景锁定图名称。
- 时间、地点、光线、地面材质、核心地标、可出现物件、禁止物件。
- 可以切机位，不能突然跳地点或跳时代。

敌人 / 怪物 / Boss 一致性：
- 当前正式锁定图和比例图。
- 体型等级、攻击方式、移动方式、弱点、禁止跑偏方向。
- Boss 未登场时不提前完整露脸；小兵群必须同设计语言但个体不同。

道具一致性：
- 当前道具状态。
- 残片就是残片，完整体就是完整体；开启就是开启，关闭就是关闭；不能混。

声音锁定：
- 本段所有发声角色必须单独列出。
- 每个发声角色必须写清声线来源 / 声音锁定文件、年龄感、音色、语速、情绪状态、机械处理或空间处理比例和混音关系。
- 当前项目已有声音规则的角色，必须调用对应声音锁定文件。
- 没有对白但有呼吸、短笑、喘息、电子提示音、扫描声、通讯声、低频启动音、怪物嘶吼或环境触发声的对象，也必须列入声音锁定。
- 声音必须像真实现场收音，不要旁白感、字幕朗读感、AI 播报感或录音棚独白感。
- 环境音和动作声可以大于对白，但不能盖住关键信息。
- 上一段尾音如何进入本段、本段尾音如何接下一段必须写清楚。
- 有对白时，按当前项目语言规则执行；如果项目要求英文配音，则实际配音只读英文对白，中文翻译只供理解，不配音、不生成字幕、不出现在画面里。
```

如果提示词里没有这些段落，视为工作流未执行完成。

## 第 7 步：多格分镜参考图专项流程

多格分镜参考图不是普通补图，不能在缺图补图门里顺手生成。它是完整 15 秒提示词、所有主锁定图和连续性规则全部确认之后的专项生成流程。

进入条件：

```text
人物 / 角色锁定图已确认。
场景锁定图已确认。
当前敌人 / 怪物 / Boss 正式锁定图已确认。
当前敌人 / 怪物 / Boss 比例图已确认。
载具 / 武器 / 任务道具锁定图已确认。
道具状态图已确认。
声音锁定和字幕规则已写入完整提示词。
连续性四帧已确认。
所有缺图已补齐并通过 QA。
图片绑定卡已更新为真实上传顺序。
```

专项流程：

```text
1. 重新输出“新版本完整 15 秒提示词”。
   - 新版提示词必须整合所有已确认锁定图、最新 @图片N 顺序、连续性四帧、声音锁定和动作地理。
   - 不能拿补图前的旧提示词直接生成分镜参考图。

2. 写“多格分镜参考图一致性生图提示词”。
   - 逐张说明输入参考图：哪张锁场景、哪张锁主角、哪张锁伙伴 / 副角色、哪张锁载具、哪张锁敌人 / 怪物 / Boss、哪张锁比例、哪张锁道具。
   - 写清每一格对应新版完整提示词里的哪个时间段和镜头事件。
   - 写清分镜图只能学习动作顺序、镜头节奏、站位、方向、起点、落点和结尾承接。
   - 写清分镜图不能改写人物脸、伙伴特殊结构、载具结构、敌人 / 怪物 / Boss 物种、真实比例、道具状态和场景地标。

3. 实际把所有主锁定图作为参考输入放进生图工具。
   - 不能只用文字生成正式多格分镜参考图。
   - 如果工具不支持参考图输入，生成结果只能算临时草图，不能进入最终上传图包。

4. 按新版完整提示词生成对应格数多格分镜参考图。
   - 6 个细分镜头生成 6 格。
   - 7 个细分镜头生成 7 格。
   - 8 个细分镜头生成 8 格。
   - 禁止把 7-8 个镜头压缩成 6 格。

5. 生成后进入分镜图 QA。
   - QA 通过后，多格分镜参考图才能作为最后一张 @图片N 回填。
   - QA 不通过，错误图必须隔离，不得作为下一版输入参考。
```

硬规则：

```text
多格分镜参考图只能排在最后一个图片位。
多格分镜参考图不是人物、场景、敌人 / 怪物 / Boss、比例或道具主锁定图。
旧错图、废稿图、跑偏图不能参与下一版生成。
上一段最后一帧 / 尾帧承接图 / 伪真尾帧图不加入多格分镜参考图输入。
```

## 第 8 步：分镜参考图权限边界

分镜图只能锁：

```text
动作顺序
镜头节奏
站位
方向
起点和落点
结尾承接
```

分镜图不能锁：

```text
人物脸
伙伴 / 副角色的特殊结构
载具结构
敌人 / 怪物 / Boss 物种
敌人 / 怪物 / Boss 真实比例
道具状态
场景地标
```

这些必须以前面的正式锁定图为准。

## 第 9 步：分镜图生成后 QA

必须检查：

```text
是否对应正确格数。
是否一格对应一个细分镜头。
是否有字幕、标题、水印、对白气泡。
是否复制角色。
是否把普通敌人变成 Boss，或把 Boss 弱化成普通敌人。
是否让分镜图覆盖了人物 / 场景 / 敌人 / 怪物 / Boss / 道具锁定。
是否把旧版角色、旧版敌人、旧版场景带回来。
小兵 / 杂兵群是否出现同一张脸、同一个头、同一套表情复制。
```

不通过则返工，不允许直接回填。

## 第 10 步：最终交付包

最终交付必须包含：

```text
最终上传图包路径
可复制提示词文件
图片1-N 上传顺序
当前采用版本
已废弃旧版
防错审计
分镜图权限说明
```
